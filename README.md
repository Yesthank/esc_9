3. **사이퍼 원문은 퍼블릭도메인**: 북 사이퍼의 시 원문은 김소월 「진달래꽃」(시집 『진달래꽃』, 1925 — 김소월 1934년 몰, 퍼블릭도메인)입니다. 표기는 1925년 원문을 기반으로 현행 맞춤법만 기계적으로 전환(업시→없이·고히→고이·노힌→놓인·흘니→흘리·옛한글 자형 현행화)했으며, 원문 어미 '-우리다' 계열은 보존했습니다.
4. **비상업·비공식**: 본 게임은 무료·비상업 팬 창작물이며, 원작자·출판사와 무관하고 어떠한 공식 연계도 없습니다.
5. **권리자 요청 시 즉시 조치**: 권리자의 요청이 있을 경우 본 저장소의 GitHub Issues 를 통해 연락 주시면 즉시 조치(수정·비공개 전환 등)하겠습니다.
